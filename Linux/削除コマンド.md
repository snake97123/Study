# ファイルやコマンドの削除に関しての方法について記載。

### ファイルの削除
```bash
rm ファイル名
```

### カレントディレクトリのファイルを全部削除
```bash
rm *
```

### 確認をしながらファイルを削除
```bash
rm -i ファイル名
```

### メッセージを表示しながら削除
```bash
rm -v ファイル名
```

### ディレクトリの削除
```bash
rm ディレクトリ名
```
