# Xcodeに関するショートカットコマンドをまとめる
・command ＋ N 新しいファイルの作成<br>
・optionを押しながらドラッグ＆ドロップ  ボタンやラベルの複製

## ウインドウの表示に関して
・command ＋ shift + y 下のウインドウの表示<br>
・command + option + 0 右のウインドウの表示<br>
・command + 0 左のウインドウの表示

## マルチカーソル
・control + shift + 矢印<br>
・control + shift + クリック

## 拡大・縮小
・command + - 縮小<br>
・command + shift + + 拡大
